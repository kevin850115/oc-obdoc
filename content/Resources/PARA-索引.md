# PARA 知识管理系统

本文件库采用 **PARA 方法** 组织：

## 📁 文件夹结构

| 文件夹 | 说明 |
|--------|------|
| **Projects** | 正在进行的具体项目，有明确目标和截止日期 |
| **Areas** | 需要持续关注的责任领域，无明确结束日期 |
| **Resources** | 未来可能有用的主题或兴趣 |
| **Archives** | 已完成或不活跃的项目/内容 |
| **Inbox** | 临时收集，待处理的内容 |
| **Daily** | 每日笔记/日记 |

## 🔖 标签系统

- #项目/进行中 - 活跃项目
- #领域 - 责任领域
- #资源 - 参考资料
- #归档 - 已完成内容
- #收件箱 - 待处理
- #日记 - 每日记录
- #AI - 人工智能相关
- #域名 - 域名相关
- #微信机器人 - 企业微信/微信机器人
- #阿里云 - 阿里云产品/服务
- #分析报告 - 各类分析报告
- #开发 - 技术开发

## 📝 使用说明

1. 新收集的内容先放入 **Inbox**
2. 定期整理 Inbox，分类到对应文件夹
3. 项目完成后移入 **Archives**
4. 每天在 **Daily** 创建日记
