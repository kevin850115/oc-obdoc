## 🎨 UI 设计相关 Skill 详解

### 1. web-design-guidelines

**功能：**
- 审查 UI 代码是否符合最佳实践
- 检查可访问性（Accessibility）
- 审核用户体验（UX）
- 对比行业标准

**使用场景：**
```
"审查我的首页代码"
"检查这个按钮的设计是否合理"
"帮我审核整个网站的 UX"
```

---

### 2. unocss

**功能：**
- 原子化 CSS 引擎
- Tailwind CSS 兼容
- 支持自定义规则
- 支持图标、排版等预设

**使用场景：**
```
"帮我用 UnoCSS 写一个卡片组件"
"配置一个自定义主题"
"创建快捷方式 shortcuts"
```

**示例代码：**
```html
<div class="bg-blue-500 text-white p-4 rounded-lg shadow-lg hover:bg-blue-600">
  按钮
</div>
```

---

### 3. vercel-react-best-practices

**功能：**
- React 性能优化
- 组件最佳实践
- 状态管理建议
- 渲染优化

**使用场景：**
```
"优化这个 React 组件的性能"
"检查是否有不必要的渲染"
"帮我重构这个组件"
```

---

### 4. nextjs-app-router-patterns

**功能：**
- Next.js 14+ App Router 模式
- Server Components 使用
- 数据获取优化
- 路由管理

**使用场景：**
```
"创建一个 Next.js 页面"
"实现服务端渲染"
"优化数据获取"
```
