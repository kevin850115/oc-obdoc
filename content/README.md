# 我的 Obsidian 笔记库

基于 **PARA 方法** 组织的个人知识管理系统。

## 📁 目录结构

| 目录 | 用途 |
|------|------|
| **Inbox/** | 临时收集箱，快速捕捉想法和待整理内容 |
| **Daily/** | 每日日记，记录当天的想法、完成事项和反思 |
| **Projects/** | 进行中的项目，有明确目标和截止日期的任务 |
| **Areas/** | 责任领域，需要持续关注的方面（健康、财务、工作等） |
| **Resources/** | 参考资料、学习笔记、模板等 |
| **Archives/** | 已完成的项目和归档内容 |

## 🚀 使用建议

1. **快速捕捉** → 先扔进 `Inbox/`
2. **每日记录** → 在 `Daily/` 写日记
3. **定期整理** → 把 Inbox 的内容分发到对应目录
4. **项目归档** → 完成后移到 `Archives/`

## 📝 模板

- 参考 `Resources/笔记模板.md`
- 日记模板在 `Daily/` 目录下
