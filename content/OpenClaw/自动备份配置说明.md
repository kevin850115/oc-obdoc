## 🔒 脱敏处理

### 自动隐藏的敏感信息

以下信息会在备份时自动替换为 `***REDACTED***`：

| 类型 | 字段示例 | 脱敏后 |
|------|---------|--------|
| **API Keys** | `apiKey` | `***REDACTED***` |
| **Client Secrets** | `clientSecret` | `***REDACTED***` |
| **Tokens** | `token` | `***REDACTED***` |
| **加密密钥** | `encodingAESKey` | `***REDACTED***` |
| **私钥** | `privateKeyPem` | `***REDACTED***` |

### 保留的非敏感信息

以下信息会正常显示：

| 类型 | 字段示例 |
|------|---------|
| **端点 URL** | `baseUrl`, `webhookPath` |
| **配置选项** | `port`, `mode`, `enabled` |
| **公开 ID** | `clientId`, `deviceId` |
| **版本信息** | `version`, `model` |

## 🤖 自动化脚本

### 脚本位置
```
/home/admin/.openclaw/workspace/obsidian-docs/scripts/backup-openclaw-config.sh
```

### Cron 配置
```bash
0 2 * * 0 /home/admin/.openclaw/workspace/obsidian-docs/scripts/backup-openclaw-config.sh >> /tmp/openclaw-backup.log 2>&1
```

**解释：**
- `0 2 * * 0` = 每周日 凌晨 2:00
- 输出重定向到 `/tmp/openclaw-backup.log`

## 📝 恢复配置

由于备份已脱敏，恢复配置需要从服务器本地获取：

```bash
# 1. 停止 OpenClaw 服务
openclaw gateway stop

# 2. 从服务器本地备份恢复
# 完整配置存储在：/home/admin/.openclaw/openclaw.json

# 3. 重启服务
openclaw gateway start
```

**建议：** 定期在安全的本地位置备份完整配置（如加密的外部存储）。
