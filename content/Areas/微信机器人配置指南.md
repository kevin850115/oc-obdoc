## 方案一：个人微信

### 适用场景

- ✅ 个人 AI 助手
- ✅ 小范围测试
- ✅ 私人聊天机器人

### 前置要求

- 需要购买第三方服务的 API Key
- 需要可公网访问的服务器
- 需要代理服务器

### 步骤 1: 安装插件

```bash
openclaw plugins install @canghe/openclaw-wechat
```

### 步骤 2: 获取 API Key

联系插件客服购买 API Key（付费服务）

**注意:** 这是第三方服务，需要付费购买

### 步骤 3: 配置插件

```bash
# 设置 API Key（必需）
openclaw config set channels.wechat.apiKey "wc_live_xxxxxxxxxxxxxxxx"

# 设置代理 URL（必需）
openclaw config set channels.wechat.proxyUrl "http://your-proxy-server:3000"

# 设置 Webhook 主机（云服务器必需）
openclaw config set channels.wechat.webhookHost "your-server-ip"

# 启用通道
openclaw config set channels.wechat.enabled true
```

### 步骤 4: 完整配置（可选）

编辑配置文件 `~/.openclaw/openclaw.json`:

```json
{
  "channels": {
    "wechat": {
      "enabled": true,
      "apiKey": "wc_live_xxxxxxxxxxxxxxxx",
      "proxyUrl": "http://your-proxy:3000",
      "webhookHost": "1.2.3.4",
      "webhookPort": 18790,
      "webhookPath": "/webhook/wechat",
      "deviceType": "mac"
    }
  }
}
```

### 步骤 5: 扫码登录

```bash
openclaw gateway restart
```

首次启动会显示二维码，用微信扫描登录。

### 多账号配置

```json
{
  "channels": {
    "wechat": {
      "accounts": {
        "work": {
          "apiKey": "wc_live_work_xxx",
          "webhookHost": "1.2.3.4"
        },
        "personal": {
          "apiKey": "wc_live_personal_xxx",
          "webhookHost": "1.2.3.4"
        }
      }
    }
  }
}
```

## 方案二：企业微信（推荐）

### 适用场景

- ✅ 企业内部使用
- ✅ 客服机器人
- ✅ 团队协作

### 优势

- ✅ 官方支持，稳定可靠
- ✅ 免费使用
- ✅ 支持群聊
- ✅ 支持富文本消息

### 步骤 1: 安装插件

```bash
openclaw plugins install @sunnoy/openclaw-plugin-wecom
```

### 步骤 2: 企业微信后台配置

#### 2.1 创建应用

1. 登录 [企业微信管理后台](https://work.weixin.qq.com/)
2. 进入「应用管理」
3. 点击「创建应用」
4. 填写应用信息（名称、图标等）

#### 2.2 配置接收消息服务器

在应用详情页：

- **URL:** `http://your-server-ip:18790/webhook/wecom`
- **Token:** 自定义（如 `mytoken123`）
- **EncodingAESKey:** 随机生成

保存配置，记下：
- Corp ID
- Agent ID
- Secret
- Token
- EncodingAESKey

### 步骤 3: 配置插件

```bash
# 进入配置模式
openclaw configure --section channels

# 选择 wecom，按提示输入：
# - Corp ID
# - Agent ID  
# - Secret
# - Token
# - EncodingAESKey
```

### 步骤 4: 完整配置

编辑 `~/.openclaw/openclaw.json`:

```json
{
  "channels": {
    "wecom": {
      "enabled": true,
      "corpId": "wwxxxxxxxxxxxx",
      "agentId": 1000001,
      "secret": "xxxxxxxxxxxx",
      "token": "mytoken123",
      "encodingAesKey": "xxxxxxxxxxxx",
      "webhookPort": 18790,
      "webhookPath": "/webhook/wecom"
    }
  }
}
```

### 步骤 5: 重启网关

```bash
openclaw gateway restart
```

### 测试

在企业微信中给机器人发送消息，应该能收到回复。

## 方案三：微信公众号

### 适用场景

- ✅ 公众号自动回复
- ✅ 粉丝互动
- ✅ 内容推送

### 步骤 1: 安装插件

```bash
openclaw plugins install @ianShaw027/wemp
```

### 步骤 2: 微信公众号后台配置

1. 登录 [微信公众号平台](https://mp.weixin.qq.com/)
2. 进入「设置与开发」→「基本配置」
3. 配置服务器：
   - **URL:** `http://your-server-ip:18790/webhook/wemp`
   - **Token:** 自定义
   - **EncodingAESKey:** 随机生成

### 步骤 3: 配置插件

```bash
openclaw configure --section channels
# 选择 wemp，按提示输入配置
```

### 步骤 4: 完整配置

```json
{
  "channels": {
    "wemp": {
      "enabled": true,
      "appId": "wxXXXXXXXXXXXX",
      "appSecret": "xxxxxxxxxxxx",
      "token": "mytoken123",
      "encodingAesKey": "xxxxxxxxxxxx",
      "webhookPort": 18790
    }
  }
}
```

### 步骤 5: 重启网关

```bash
openclaw gateway restart
```

## ⚠️ 常见问题

### 问题 1: 机器人收不到消息

**检查项:**
- ✅ webhookHost 是否配置为公网 IP
- ✅ webhookPort 是否开放（防火墙）
- ✅ 网关是否正常运行：`openclaw gateway status`
- ✅ 服务器是否能被外网访问

**解决:**
```bash
# 检查网关状态
openclaw gateway status

# 查看日志
openclaw gateway logs

# 测试端口
telnet your-server-ip 18790
```

---

### 问题 2: 二维码扫描后无法登录

**原因:**
- 第三方服务问题
- 网络问题
- 账号限制

**解决:**
1. 检查网络连接
2. 联系 API Key 提供商
3. 尝试重新扫码

---

### 问题 3: 企业微信回调地址校验失败

**原因:**
- Token 不匹配
- EncodingAESKey 错误
- URL 不可访问

**解决:**
```bash
# 1. 确认企业微信后台配置
# 2. 确认插件配置一致
# 3. 测试 URL 可访问性
curl http://your-server-ip:18790/webhook/wecom
```

---

### 问题 4: 消息回复延迟

**原因:**
- 网络延迟
- 模型响应慢
- 服务器性能

**解决:**
1. 使用更快的模型
2. 优化服务器配置
3. 检查网络连接

---

### 问题 5: 无法发送图片

**检查:**
- 是否支持图片消息
- 图片格式是否正确
- 图片大小是否超限

**解决:**
```json
{
  "channels": {
    "wechat": {
      "enableImages": true,
      "maxImageSize": "5MB"
    }
  }
}
```
